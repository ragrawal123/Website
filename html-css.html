HTML vs CSS:

HTML

HTML is the language that software engineers use to develop webpages. It serves as the base for the mechanics of the page, such as the links you see or the navbar above. HTML helps develop just pure text, and it can also be used to create user interfaces for desktop apps like Spotify. In fact, as you probably already know, this entire website was a prokect that my group did to learn HTML. If you right click and view the page source, you can see the HTML code for all the pages.

CSS

CSS is a helper language that software engineers use to design webpages. It serves as the language that would create the design of the webpage. So all the positioning and colors and font you see on this webpage and others too is mostly the work of CSS. Like HTML, it can also be used to create user interfaces for desktop apps like Spotify.

Documentation:

HTML

HTML can be hard, though not as difficult as CSS in my opinion. So, having some resources at your disposal any time can be very useful when learning HTML or trying to code it. Here are some sources that I found useful when coding the website:
    1. W3Schools HTML
    2. Learn ShayHowe HTML
    3. Mimo
The upper three links are sources that we read to start learning the basics of HTML as well as a reference sheet, to look at different HTML tags and see what they do and how they work. The W3Schools site is especially good to act as a HTML documentation site while the Mimo app on our phones was useful to learn some of the basics of HTML.

CSS

CSS was the hardest part of designing this website. Making sure that every text or heading or link is positioned correctly did make us frustrated at times. But, we had some documentations and sources below that we found useful:
    1. W3Schools CSS
    2. CSS Cheat Sheet
    3. Mimo
When starting to learn CSS, we mostly used the Mimo app on my phone to learn the basics, but we found it slow and not that helpful. So we turned to W3Schools for documentation and the CSS Cheat Sheet for tips. Another hard part about this was designing the website out of thin air and coding it. To get an idea of what we wanted the website to look like, we used Figma to design the website and then code from there.

Learning:

To get started on learning the basics of HTML and CSS, we'd recommend:
    1. Mimo
    2. W3School
    3. CodeAcademy-HTML
    4. CodeAcademy-CSS
All these sites are good for learning HTML and CSS, and they also provide exercises for you to go through to test your knowledge on HTML and CSS. However, the best practice comes from trying to make a webpage like we did, so once you feel comfortable try making a webpage! It will require a lot of trial and error and more research, but that's the purpose of the ATCS class.
